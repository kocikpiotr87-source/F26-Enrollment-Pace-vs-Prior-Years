How to read this. Each line is a fall term's enrolled headcount reconstructed as of every day before that term, plotted against days until term start — so curves align on recruitment stage, not calendar date. The dashed line marks F26's current position (53 days out); its curve stops there because data only runs through July 06, 2026. Enrolled-as-of-date is reconstructed from i805 add/drop dates. Term start = modal class start date (i805 C0200). Counts are total enrolled (all career/program types).

All figures are the enrolled population at the same point in each cycle (53 days before term start). The Δ column is F26 minus F25. F26 is taken from the official daily OMNI file (matching the F26 enrollment dashboard); prior years are reconstructed from i805 at the same stage.
